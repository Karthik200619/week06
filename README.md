# Week-6 Assignments

This repository contains all the assignments for Week 6. Each assignment is placed in its respective folder and demonstrates various HTML and CSS skills, focusing on layouts like Flexbox and CSS Grid.

## Assignments

- **[assignment-1](./assignment-1)**: Basic HTML layout featuring an introduction, card sections, and a footer.
- **[assignment-2.1](./assignment-2.1)**: A responsive Apple Grid layout showcasing different products and features using CSS Grid.
- **[assignment-2.2](./assignment-2.2)**: A layout for a food spot, containing different sections for top navigation, middle banners, and bottom informational rows.
- **[assignment3](./assignment3)**: A Travel Landing Page with sections for outdoor activities and types of tourism, designed with a grid layout.
- **[assignment-4](./assignment-4)**: Another variation of the Apple Grid layout with a focus on specific grid positioning.
- **[assignment-5](./assignment-5)**: A testimonials grid layout displaying reviews from different users.

Feel free to explore each folder for more specific details about each assignment.
